# DeFi Savings Smart Contract

## Overview
This is a Solidity smart contract that allows users to securely deposit and withdraw ETH with a time-lock mechanism.

## Features
- Deposit ETH
- Withdraw ETH after a lock period
- Reentrancy protection
- Event logging for transactions

## Technologies Used
- Solidity
- Ethereum

## How It Works
- Users deposit ETH into the contract
- Funds are locked for a specific time
- Users can withdraw after the lock period
